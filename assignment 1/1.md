# Question 1 – String

 Write a JavaScript function to reverse a string without using the built-in reverse() method.

# Example:

Input:

hello


Output:

olleh


# Solution:

**Step 1:** Define variable 'text' with value 'hello'

**Step 2:** Create global variable 'reversed' with value ""

**Step 3:** Introduce a for loop which iterates through each letter of 'text', so from 1->5

**Step 4:** Inside for loop, the reversed order of each letter (for i, we would get ((length of "hello")-i)th index of "hello")

   - When i=1, (5-1)th index -> "o"
   - When i=2, (5-2)th index -> "l"
   - When i=3, (5-3)th index -> "l"
   - When i=4, (5-4)th index -> "l"
   - When i=5, (5-5)th index -> "h"

**Step 5:** We append/concat each character to 'reversed'

**Step 6:** Output 'reversed'
