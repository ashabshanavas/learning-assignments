Question 1 – String
Write a JavaScript function to reverse a string without using the built-in reverse() method.

Example:

Input:

hello
Output:

olleh

Solution:

Step 1: Define variable 'text' with value 'hello'
Step 2: Create global variable 'reversed' with value ""
Step3: Introduce a for loop which iterates through each letter of 'text', so from 1->5
Step4: Inside for loop, the reversed order of each letter (for i, we would get ((length of "hello")-i)th index of "hello")
    1. When i=1, (5-1)th index -> "o"
    2. When i=2, (5-2)th index -> "l"
    3. When i=3, (5-3)th index -> "l"
    4. When i=4, (5-4)th index -> "l"
    5. When i=5, (5-5)th index -> "h"
Step5: We append/concat each character to 'reversed'
Step6: Output 'reversed'
